Chapter 6: Programming Languages

Terms Definitions
Machine Language A computer system that learns and adapts to new information by using algorithms and statistical models
Debugging Going through a section of code to fix any errors
Assembler A program that converts programming languages into machine code
Assembly Language The programming language that computers directly use
Translator A computer program that converts human-friendly code into machine language code
Compiler A program that converts instructions into machine code
Interpreter A program that directly executes instructions written in a programming language
Cross-Platform Software Software able to be used on several different computers
Natural Language A computer programming language that can understand human language
Formal Language A computer programming language that can be understood by computers
Machine Independence Able to operate on many different types of equipment and software
High-Level Programming Language A programming language that uses natural words and punctuation to make coding easier for humans
Object-Oriented Programming (OOP) A computer programming language that organizes software around data or objects rather than functions or logic
Scripting Language A programming language that is used to manipulate and automate a system
